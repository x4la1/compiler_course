# Примеры программ на языке Init

### 1. `SumNumbers` — сложение двух чисел

```
float a;
float b;
input(a);
input(b);
float sum = a + b;
print(sum);
```

---

### 2. `CircleSquare` — площадь круга

```
float radius;
input(radius);
float area = Pi * radius ^ 2;
print(area);
```

---

### 3. `FahrenheitToCelsius` — перевод температуры

```
float fahrenheit;
input(fahrenheit);
float celsius = (fahrenheit - 32.0) * 5.0 / 9.0;
print(celsius);
```
