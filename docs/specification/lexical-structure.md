# Лексическая структура языка

## 1. Примеры кода

```
// Программа приветствия
string name;
print("Enter your name: ");
input(name);
if (name == "") {
    print("Hello, stranger!");
} else {
    print("Hello, ", name, "!");
}
```

```
// Факториал
float n;
print("Enter n: ");
input(n);
if (n < 0) {
    print("Error: n must be non-negative.");
} else {
    float result = 1;
    while (int i = 1; i <= n; i = i + 1) {
        result = result * i;
    }
    print("Factorial: ", result);
}
```

```
// Функция вычисления абсолютного значения
func abs(x: int): float {
    if (x < 0) {
        return -x;
    }
    return x;
}

float a = -17;
float b = abs(a);
print("Absolute value of ", a, " is ", b);
```

```
//Работа с логическими значениями
bool isPositive = true;
float value = -5;
if (value < 0) {
    isPositive = false;
}
if (isPositive == true) {
    print("Value is positive");
} else {
    print("Value is not positive");
}
```

## 2. Ключевые слова

| Слово      | Предназначение                          |
| ---------- | --------------------------------------- |
| `if`       | Начало условного оператора              |
| `else`     | Альтернативная ветвь условия            |
| `while`    | Цикл с предусловием                     |
| `break`    | Немедленный выход из цикла              |
| `print`    | Вывод значений                          |
| `input`    | Ввод значения                           |
| `void`     | Тип «отсутствие значения»               |
| `float`    | Тип: число с плавающей точкой           |
| `string`   | Тип: строка                             |
| `bool`     | Тип: логическое значение                |
| `func`     | Объявление функции                      |
| `return`   | Возврат значения из функции             |  
| `true`     | Литерал логической истины               |
| `false`    | Литерал логической лжи                  |
| `repeat`   | Начало цикла с постусловием             |
| `until`    | Условие завершения цикла `repeat`       |
| `switch`   | Начало множественного ветвления         |
| `case`     | Ветка в `switch`                        |
| `default`  | Ветка по умолчанию в `switch`           |

## 3. Идентификаторы

- Начинаются с буквы (`a–z`, `A–Z`) или `_`.
- Последующие символы: буквы, цифры, `_`.
- Чувствительны к регистру.
- Не могут совпадать с ключевыми словами.

## 4. Литералы

### 4.1. Литералы чисел

- Числа с плавающей точкой: `42`, `3.14`, `-17`,

### 4.2. Литералы строк

- Однострочные строки: `"Hello"`
- Многострочные строки: """..."""
- Экранирование: `\"`, `\\`, `\n`
- Строковые литералы могут содержать только **символы ASCII** (коды 0–127).

### 4.3. Литералы логического типа

- Логические литералы: `true`, `false`
  
## 5. Операторы

| Категория           | Операторы                              |
|---------------------|----------------------------------------|
| Арифметические      | `+`, `-`, `*`, `/`, `%`, `@`, `^`      |
| Сравнения           | `==`, `!=`, `<`, `<=`, `>`, `>=`       |
| Логические          | `&&`, `||`, `!`                        |
| Присваивания        | `=`                                    |
| Унарные             | `+`, `-`, `!`                          |

> Примечание:  
> - `%` — остаток от деления.

## 6. Прочие лексемы

| Лексема | Назначение |
|--------|-----------|
| `;`    | Завершение оператора |
| `,`    | Разделитель аргументов |
| `{ }`  | Блок кода |
| `( )`  | Выражения, параметры цикла/условия |
| `"`    | Начало/конец однострочной строки |
| `"""`  | Начало/конец многострочной строки |
| пробел, таб, перевод строки | Разделители |
