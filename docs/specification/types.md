# Семантика типов данных языка Init

Язык Init поддерживает **четыре типа данных**:

| Тип      | Описание                                              
|----------|------------------------------------ |
| `float`  | Числа с плавающей точкой            |
| `bool`   | Логический тип: `true` или `false`  |
| `string` | Последовательность Unicode-символов |
| `void`   | Отсутствие значения                 |

Ключевые принципы:
- Тип переменной **определяется при объявлении** и **не может меняться**.
- Все переменные **должны быть объявлены явно**.
- Присваивание допустимо **только между совместимыми типами**.
- Операторы **перегружены по типам** (например, `+` работает и для чисел, и для строк).

---

## Литералы

| Тип      | Синтаксис литерала                             | Примеры                     |
|----------|-----------------------------------------------|-----------------------------|
| `float`  | Последовательность цифр с опциональной точкой | `42`, `3.14`, `15` |
| `bool`   | Ключевые слова `true`, `false`                | `true`, `false`             |
| `string` | Строка в двойных кавычках; поддержка экранирования | `"Hello"`, `"He said: \"Hi!\""` |
| `void`   | Нет литерала                                   | —                           |

---

## Операторы

### Арифметические (`float`)
- `+`, `-`, `*`, `/` — стандартные арифметические операции
- `%` — остаток от деления
- `^` — возведение в степень
- `@` — целочисленное деление (результат приводится к `float`)

### Строковые (`string`)
- `+` — конкатенация строк  
  Пример: `"Hello" + " world"` → `"Hello world"`

### Логические (`bool`)
- `!` — логическое НЕ
- `&&` — логическое И
- `||` — логическое ИЛИ

### Сравнение (для `float`, `string`, `bool`)
- `==`, `!=` — равенство / неравенство
- Для строк — **побайтовое сравнение** (чувствительно к регистру)
- `<`, `<=`, `>`, `>=` — допустимы **только для `float`**

---

## Переменные

### Объявление
```
float x;
bool flag = true;
string name = "Alice";
```

- Тип указывается **до имени**.
- Инициализация при объявлении **необязательна**.
- Без инициализации переменная **имеет неопределённое значение**.

### Присваивание
```
x = 3.14;          
flag = x > 0;     
name = "Bob";     
x = "text"; - Ошибка: несовместимые типы
```

- Присваивание требует **типовую совместимость**.
- Результат выражения в правой части должен быть **совместим с типом переменной**.

### Область видимости
- Переменные, объявленные на верхнем уровне — **глобальные**.
- Переменные внутри блоков `{ }` — **локальные**.
- Повторное объявление в одной области видимости **запрещено**.

---

### Кодировка и ограничения строк

- Строки в языке **Init** представляют собой последовательности **символов в кодировке UTF-8**.
- Однако допускаются **только символы из набора 7-битного ASCII** (коды от 0 до 127).
- Это включает:
  - латинские буквы (`A–Z`, `a–z`),
  - цифры (`0–9`),
  - знаки препинания и спецсимволы (`!`, `"`, `#`, `$`, `%`, `&`, `'`, `(`, `)`, `*`, `+`, `,`, `-`, `.`, `/`, `:`, `;`, `<`, `=`, `>`, `?`, `@`, `[`, `\`, `]`, `^`, `_`, `` ` ``, `{`, `|`, `}`, `~`).

---

## Примечания по `void`

- Тип `void` **нельзя использовать в объявлениях переменных**.
- Он используется **только в объявлении процедур** (функций без возврата).
- В теле процедуры допустима инструкция `return;` **без выражения**.

---

## Встроенные функции
Язык предоставляет набор встроенных функций для работы со строками:
- `length(s: string): float` — возвращает длину строки
- `charAt(s: string, i: float): string` — возвращает i-й символ (нумерация с 0)
- `toLowerCase(s: string): string` — переводит строку в нижний регистр
