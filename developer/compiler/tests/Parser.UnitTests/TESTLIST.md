# Список тестов

## Класс Parser

- [x] Разбор выражения с унарным "+" `+5` -> 5.
- [x] Разбор выражения с унарным "-" `-10` -> -10.
- [x] Разбор выражения с несколькими унарными "-" `--5` -> 1.
- [x] Разбор выражения с несколькими унарными "!" (Логическое НЕ) `!!5` -> 1, `!0` -> 1, `!1` -> 0
- [x] Разбор выражения c возведением в степень `2 ^ 3` -> 8
- [x] Проверка правоассоциативности возведения в степень `2 ^ 3 ^ 2` -> 512
- [x] Разбор выражения c возведением в степень с приоритетом `-2 ^ 3` -> -8
- [x] Разбор выражения c произведением `6 * 2` -> 12
- [x] Разбор выражения c делением `9 / 2` -> 4.5
- [x] Проверка левоассоциативности деления `8 / 2 / 2` -> 2
- [x] Разбор выражения c делением по модулю `10 % 3` -> 1
- [x] Разбор выражения c делением нацело `10 @ 3` -> 3
- [x] Разбор выражения c разными приоритетами операций `1 + 2 * 3` -> 7
- [x] Разбор выражения c сложением `1 + 2` -> 3
- [x] Разбор выражения c вычитанием `5 - 2` -> 3
- [x] Проверка левоассоциативности вычитания `1 - 2 - 3` -> -4
- [x] Разбор выражения с сравнением и арифметикой  `1 + 2 < 5` -> 1
- [x] Разбор выражения с операторами сравнения `5 <= 5` -> 1, `10 > 5` -> 1, `10 >= 10"` 	-> 1, `5 < 5` -> 0, `(5 < 10) == 1` -> 1, `(5 < 10) != 0` -> 1
- [x] Разбор выражения с сравнениями и логическим оператором И `(1 == 1) && (2 != 3)` -> 1
- [x] Разбор выражения с логическими операторами И, ИЛИ `1 && 0 || 1` -> 1
- [x] Разбор выражения со скобками `(1 + 2) * 3` -> 9
- [x] Разбор выражения с константой -> `Pi * 2` -> 6.28318530718, `Euler`
- [x] Разбор арифметического выражения с операторами всех приоритетов `1 + 2 * 3 ^ 2 - 4 / 2` -> 17
- [x] Разбор выражения с операторами всех приоритетов `((10 * 5 + -2 ^ 3 ^ 2) < -400) == 1 && !0 || 0` -> 1
